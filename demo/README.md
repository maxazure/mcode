# 贪吃蛇游戏

这是一个使用Python和Pygame开发的经典贪吃蛇游戏。

## 安装依赖

在运行游戏之前，请确保安装了Pygame库：

```bash
pip install pygame
```

## 运行游戏

```bash
cd demo
python snake_game.py
```

## 游戏说明

### 操作方法
- **方向键** (↑↓←→): 控制蛇的移动方向
- **P键**: 暂停/继续游戏
- **空格键**: 游戏结束后重新开始
- **关闭窗口**: 退出游戏

### 游戏规则
1. 控制绿色的蛇吃到红色的食物
2. 每吃到一个食物，蛇会变长，分数增加10分
3. 避免撞到墙壁或自己的身体
4. 撞墙或撞到自己时游戏结束

### 游戏特性
- 流畅的蛇移动动画
- 网格化游戏界面
- 实时分数显示
- 暂停功能
- 游戏结束后可重新开始
- 防止蛇直接掉头的设计

## 技术实现

- **编程语言**: Python 3
- **游戏库**: Pygame
- **设计模式**: 面向对象编程
- **主要类**:
  - `Snake`: 蛇的行为和渲染
  - `Food`: 食物的生成和渲染
  - `Game`: 游戏主逻辑和循环
